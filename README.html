AVR Microcontroller Firmware Development Framework

Here is a framework for developing firmware for AVR single chip microcontrollers.

News

  • 1 January 2015 -- Upgraded binutils to 2.25 and gcc to 4.8.4.
  • 21 January 2015 -- Cleaned up the toolchain makefile. Build gcc prerequisites, including libiconv, from scratch.
  • 5 August 2015 -- Upgraded gcc to 4.9.3.
  • 22 January 2016 -- Upgraded gcc to 5.3.0.

  • 21 November 2016 -- Upgraded binutils to 2.27 and gcc to 6.2.0.

Note: I have largely migrated to ARM microcontrollers, so this AVR framework will only be updated sporadically from now on.

Git Repository

The source code is available at: http://git.munts.com/avr-mcu

Use the following command to clone it:

git clone http://git.munts.com/avr-mcu.git

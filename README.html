AVR Microcontroller Frameworks

Here are some firmware development frameworks for certain 8-bit AVR single chip microcontrollers. I am interested in alternatives to the traditional GCC development environment, so there are also some experiments with Arduino C++ and mikroPascal for AVR microcontrollers.

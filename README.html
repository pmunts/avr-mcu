AVR Microcontroller Firmware Development Framework

Here is a framework for developing firmware for AVR single chip microcontrollers.

News

  • 3 April 2012 -- Reorganized into subdirectories.

MCU Independent Subdirectories:

common MCU independent common library source files
include MCU independent include files
test Test and demo programs
toolchain AVR toolchain (binutils, gcc, avr-libc) builder
utilities Utility programs and scripts
